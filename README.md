# Movie Ratings Dataset Analysis

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Laraazad/movie_ratings_dataset/blob/main/statistics.ipynb)

This project explores and analyzes a movie ratings dataset using Python in Jupyter Notebook / Google Colab.

## Files Included

- `statistics.ipynb` → Python notebook containing the analysis
- `movie_ratings.csv` → Dataset used for analysis

## Topics Covered

- Load & Explore the Dataset
- Histograms
- Cross-tabulation & Pivot Tables
- Box Plots & Outlier Detection
- Count Plots
- Measures of Central Tendency & Dispersion
- Skewness & Distribution Shapes
- Z-Scores & p-Values
- Chebyshev's Rule
- t-Tests
- Confidence Intervals
- Correlation Analysis
- Scatter Plots & Pair Plot
- Distribution Plots
- Linear Regression (OLS)

## Libraries Used

- Pandas, NumPy, Matplotlib, Seaborn, SciPy, Statsmodels
